# Package and run the application

- Package

```shell
./mvnw package
```

- Run

```shell
java -jar target/quarkus-app/quarkus-run.jar
```

<br>

Notes:

If /etc/hosts entry warning -> <br>
uname -n <br>
then add the result in /etc/hosts <br>
127.0.0.1 localhost %here%
